## 🎯 WHAT ARE BUG COMMANDS?

Bug commands send MANY messages to someone very fast. This can crash their WhatsApp or make their phone slow.

---

## 📱 HOW TO USE

### Step 1: Get Phone Number
- Find the person's WhatsApp number (with country code)
- Example: 2349054345858 (Nigeria number)

### Step 2: Choose Command

### 🪲 WEAK ATTACK (Safe for testing)

- Sends 100 annoying messages
- Not too dangerous

### 💀 MEDIUM ATTACK (Can cause problems)
